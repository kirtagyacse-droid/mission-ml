01 · The Student's t-Test Overview

The t-test is used to compare means when the population standard deviation ($\sigma$) is unknown. It relies on the t-distribution and is widely used for smaller samples.

02 · One-Sample t-Test

Compares the mean of a single sample to a hypothesized population mean ($\mu_0$).

One-Sample t-Statistic
t = (X̄ − μ₀) / (s / √n)

03 · Independent Two-Sample t-Test

Compares the means of two independent groups (e.g., comparing test scores of two different classrooms).

Independent t-Statistic (Equal Variances)
t = (X̄₁ − X̄₂) / [s_p · √(1/n₁ + 1/n₂)] where s_p = √[((n₁−1)s₁² + (n₂−1)s₂²) / (n₁ + n₂ − 2)]

04 · Paired Sample t-Test

Compares means from the same group at different times (e.g., weights before and after a diet program). We analyze the differences $D_i = X_{1,i} - X_{2,i}$:

Paired t-Statistic
t = D̄ / (s_d / √n)

05 · Degrees of Freedom (df)

Degrees of freedom represent the number of independent values that can vary in an analysis:

  • One-sample / Paired t-test: $df = n - 1$
  • Independent two-sample t-test: $df = n_1 + n_2 - 2$

06 · Assumptions of t-Tests

For t-test results to be valid, the following assumptions must hold:

  1. Scale: Continuous or interval scale of measurement.
  2. Independence: Random sampling with independent observations.
  3. Normality: The populations from which samples are drawn should be approximately normal.
  4. Homoscedasticity: Independent groups should have approximately equal variances (only for two-sample t-tests).

07 · Normality and Homoscedasticity Checks

Use visual checks (Q-Q plots) and formal statistical tests:

  • Normality: Shapiro-Wilk test (scipy.stats.shapiro).
  • Equal Variance: Levene's test (scipy.stats.levene).

08 · Python Code Implementation

import numpy as np
from scipy import stats

group_a = [23, 25, 28, 32, 24, 27, 29, 31, 25, 26]
group_b = [30, 32, 35, 31, 29, 33, 34, 32, 31, 30]

# Perform independent two-sample t-test
t_stat, p_val = stats.ttest_ind(group_a, group_b, equal_var=True)
print(f"t-statistic: {t_stat:.4f}, p-value: {p_val:.4f}")

09 · Quick Reference Card

🚀 t-Test Quick Reference

  • One-sample: Compare one group mean to a target value.
  • Independent: Compare means of two distinct groups.
  • Paired: Compare means of the same group under two conditions.
  • Welch's t-test: Use equal_var=False in Python when group variances differ.
