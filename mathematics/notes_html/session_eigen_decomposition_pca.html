01 · Matrix Diagonalization & Eigendecomposition

If an $n imes n$ matrix $A$ has $n$ linearly independent eigenvectors, it can be factored into a product of matrices:

Eigendecomposition
A = P · D · P⁻¹ where P's columns are eigenvectors and D is diagonal with eigenvalues.

02 · Symmetric Matrices & Spectral Theorem

The **Spectral Theorem** states that any real symmetric matrix $A$ can be orthogonally diagonalized:

Orthogonal Diagonalization
A = Q · D · Qᵀ (since Qᵀ = Q⁻¹)

03 · PCA Problem Formulation

Principal Component Analysis (PCA) is a dimensionality reduction technique. The goal is to project data onto a lower-dimensional subspace while maximizing the variance of the projected points.

04 · Covariance Matrix Derivation

PCA computes the eigenvectors of the **covariance matrix** $\Sigma$ of the centered data matrix $X$:

Covariance Matrix
Σ = (1 / n) · XᵀX

The eigenvectors of $\Sigma$ represent the principal directions of variance, and the eigenvalues denote the magnitude of variance in those directions.

05 · PCA Algorithm Steps

  1. Standardize (center and scale) the dataset.
  2. Compute the covariance matrix $\Sigma$.
  3. Compute eigenvalues and eigenvectors of $\Sigma$.
  4. Sort eigenvalues in descending order.
  5. Select the top $k$ eigenvectors to form the projection matrix $W$.
  6. Transform the original data: $Y = XW$.

06 · Reconstruction Error & Variance Explained

The proportion of variance explained by the top $k$ principal components is:

Explained Variance Ratio
Explained Variance = Σ_{i=1}^k λ_i / Σ_{j=1}^d λ_j

07 · PCA Variants (Kernel & Incremental)

  • Kernel PCA: Uses the kernel trick to perform non-linear dimensionality reduction.
  • Incremental PCA: Processes data in mini-batches, suitable for large datasets that do not fit in RAM.

08 · Python PCA Implementation

from sklearn.decomposition import PCA
import numpy as np

X = np.random.randn(100, 5)

pca = PCA(n_components=2)
X_reduced = pca.fit_transform(X)

print("Explained Variance Ratio:", pca.explained_variance_ratio_)

09 · Quick Reference Card

🚀 PCA & Eigendecomposition Cheat Sheet

  • Eigendecomposition: $A = PDP^{-1}$.
  • PCA objective: Maximize projected variance / minimize reconstruction error.
  • Scree Plot: Plot of eigenvalues to help choose the number of principal components.
