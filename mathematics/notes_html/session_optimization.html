01 · Introduction to Mathematical Optimization

Optimization is the process of finding the inputs that minimize or maximize an objective function (e.g., minimizing a loss function in machine learning).

02 · Convexity & Convex Optimization

A function is **convex** if a line segment drawn between any two points on the graph lies above or on the graph. Convex functions are easier to optimize because any local minimum is guaranteed to be the global minimum.

03 · Global vs Local Extrema

  • Global Minimum: The absolute lowest value of the function over its entire domain.
  • Local Minimum: The lowest value of the function within a specific local neighborhood.

04 · Saddle Points & Plateau Regions

A **saddle point** is a point where the function's gradient is zero, but it is not a local extremum (it curves up in one direction and down in another). Saddle points are major obstacles when training deep neural networks.

05 · Gradient Descent Algorithm

Gradient Descent is an iterative optimization algorithm that updates parameters in the direction of the steepest descent (opposite to the gradient $ abla f$):

Gradient Descent Step
θ_(new) = θ_(old) − η · ∇f(θ_(old)) where η is the learning rate.

06 · Learning Rate Optimization

  • Too small learning rate: Very slow convergence.
  • Too large learning rate: Overshoots the minimum and may diverge.

08 · Modern Optimizers (Adam, RMSprop)

Modern deep learning optimizers adapt the learning rate for each parameter using running averages of the gradients (momentum) and squared gradients (RMSprop).

09 · Quick Reference Card

🚀 Optimization Cheat Sheet

  • Gradient Direction: $ abla f( heta)$ points in direction of steepest ascent.
  • Convex Function: Local minimum = Global minimum.
  • SGD: Updates parameters using a single random sample at a time.
