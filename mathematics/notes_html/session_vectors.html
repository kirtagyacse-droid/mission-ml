01 · Geometric vs Algebraic Representation

Geometric: A vector is an arrow in space with **magnitude** (length) and **direction**.

Algebraic: An ordered list of numbers represented as a column vector $\mathbf{v} = [v_1, v_2, ..., v_n]^T$.

02 · Basic Vector Operations

  • Vector Addition: $\mathbf{u} + \mathbf{v} = [u_1+v_1, u_2+v_2]^T$. Geometrically represented by head-to-tail addition.
  • Scalar Multiplication: $c\mathbf{v} = [cv_1, cv_2]^T$. Scales the vector's length without changing its line of action.

03 · Dot Product (Algebraic & Geometric)

Dot Product Formulas
Algebraic: u · v = Σ uᵢ · vᵢ Geometric: u · v = ||u|| · ||v|| · cos(θ)

If $\mathbf{u} \cdot \mathbf{v} = 0$, the vectors are orthogonal (perpendicular).

04 · Vector Projections

The projection of vector $\mathbf{u}$ onto vector $\mathbf{v}$ yields the shadow of $\mathbf{u}$ along the direction of $\mathbf{v}$:

Vector Projection Formula
proj_v(u) = [ (u · v) / ||v||² ] · v

05 · Vector Norms (L1, L2, L-infinity)

Norms measure the length/magnitude of vectors:

  • $L_1$ Norm (Manhattan): $||\mathbf{x}||_1 = \sum |x_i|$. Used in Lasso regularization.
  • $L_2$ Norm (Euclidean): $||\mathbf{x}||_2 = \sqrt{\sum x_i^2}$. Used in Ridge regularization.
  • $L_\infty$ Norm (Chebyshev): $||\mathbf{x}||_\infty = \max |x_i|$.

06 · Linear Combinations & Span

A **linear combination** of vectors is a sum of scaled vectors: $\mathbf{w} = c_1\mathbf{v_1} + c_2\mathbf{v_2}$. The **span** of a set of vectors is the set of all possible linear combinations that can be formed from them.

07 · Linear Independence

A set of vectors is linearly independent if no vector in the set can be written as a linear combination of the others. None of them are redundant.

08 · Basis Vectors

A **basis** for a vector space is a set of linearly independent vectors that span the entire space. Standard basis in $\mathbb{R}^2$ is $\mathbf{i} = [1, 0]^T$ and $\mathbf{j} = [0, 1]^T$.

09 · Quick Reference Card

🚀 Vectors Cheat Sheet

  • Orthogonality: $\mathbf{u} \cdot \mathbf{v} = 0$.
  • Euclidean length ($L_2$ norm): $\sqrt{\mathbf{u} \cdot \mathbf{u}}$.
  • Unit Vector: Vector with norm = 1, computed as $\mathbf{v} / ||\mathbf{v}||$.
