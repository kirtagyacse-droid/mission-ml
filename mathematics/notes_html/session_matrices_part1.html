01 · Matrix Definition & Types

A matrix is a 2D grid of numbers. Types include:

  • Diagonal Matrix: Non-zero elements only along the main diagonal.
  • Identity Matrix ($I$): Diagonal matrix with 1s on the diagonal. Acts as the number 1 for matrices.
  • Symmetric Matrix: Transpose of the matrix equals the matrix itself: $A = A^T$.

02 · Matrix Multiplication (Algebraic View)

Multiplication $C = AB$ is defined when the number of columns in $A$ matches rows in $B$. Element $C_{i,j}$ is the dot product of Row $i$ of $A$ and Column $j$ of $B$.

Matrix Multiplication Element
C_(i,j) = Σ_k A_(i,k) · B_(k,j)

03 · Matrix Multiplication (Geometric View)

Geometrically, multiplying a vector by a matrix ($A\mathbf{x}$) represents a **linear transformation** of the vector. The columns of $A$ define where the standard basis vectors land after transformation.

04 · Determinant & Volume Scaling

The **determinant** of a matrix, $\det(A)$, measures how much area (in 2D) or volume (in 3D) scales after linear transformation. If $\det(A) = 0$, the matrix squishes space into a lower dimension.

05 · Matrix Inverse & Identity

The inverse of matrix $A$ reverses the linear transformation:

Matrix Inverse Property
A · A⁻¹ = A⁻¹ · A = I

An inverse exists only if $\det(A) eq 0$ (the matrix is non-singular).

06 · Transpose & Symmetric Matrices

The transpose $A^T$ is formed by swapping rows with columns: $(A^T)_{i,j} = A_{j,i}$.

07 · Orthogonal Matrices

An orthogonal matrix preserves lengths and angles, representing pure rotations or reflections:

Orthogonal Matrix Property
Qᵀ · Q = Q · Qᵀ = I → Qᵀ = Q⁻¹

08 · Python Matrix Operations

import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix Multiplication
C = np.dot(A, B)  # or A @ B
det_A = np.linalg.det(A)
inv_A = np.linalg.inv(A)

print(f"det(A): {det_A:.4f}")

09 · Quick Reference Card

🚀 Matrices Cheat Sheet

  • Matrix multiplication is non-commutative: $AB eq BA$ in general.
  • Transpose product rule: $(AB)^T = B^T A^T$.
  • Inverse product rule: $(AB)^{-1} = B^{-1} A^{-1}$.
  • Singular Matrix: $\det(A) = 0$ (no inverse exists).
